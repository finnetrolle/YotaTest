Описание интерфейса

GET /document

Запрос возвращает список хранящихся в системе документов без подгрузки их свойств

GET /document/{id}

Запрос возвращает заполненный свойствами документ по идентификатору id

GET /download/{id}

Запрос возвращает XML файл, представляющий документ с запрашиваемым id

POST /upload

Запрос позволяет загрузить на сервер файл в формате XML, прочитать его и сохранить в БД
